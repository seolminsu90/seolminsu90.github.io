개발 경험 소개

개발 경력 초기에는 주로 SNS나 웹툰 서비스 등 일반적인 웹 개발과, 앱 서비스 백엔드API 그리고 관리자 운영 페이지 등을 개발하였고 PG사 연동을 통해 PC결제를 연동한 경험도 있습니다. 그리고 몇 년의 경험을 거쳐 낚시/색칠/퍼즐과 같은 캐쥬얼 게임의 백엔드 API 개발 및 인앱결제, 인증, 푸시 서비스 및 운영툴 개발, 이벤트 페이지 개발 같은 게임 공통 플랫폼 업무도 맡았습니다. 작은 회사였던 만큼 웹 개발 및 서버 개발과 DB 설계, AWS 운용 등 여러가지 파트의 업무를 다방면으로 경험했던 장점이 있었습니다.

그 다음 회사에서는 커뮤니티 운영자들의 커뮤니티 대응 업무를 지원하기 위한 통합 운영 지원 툴을 개발하였습니다. 이 프로젝트에선 여러 커뮤니티 사이트의 게시글을 수집하기 위한 Selenium 기반의 크롤러, 커뮤니티 이슈 처리 및 동향을 관리할 수 있는 이슈 관리 파트를 맡아 개발 하였습니다. 그리고 오래되었던 게임 커뮤니티 사이트를 리뉴얼 하는 업무를 하기도 했습니다.이때부터는 Java Spring 기반의 개발 뿐 아니라 Vuejs 기반의 프론트엔드 개발도 같이 담당했습니다.

그 후에는 게임 플랫폼 개발 부서에서 게임 운영툴, 게임 QA지원 툴, 게임 이벤트 관리 툴,인앱 결제 검증 모듈, 게임 로그 수집기, 게임 데이터를 관리하는 데이터 관리 툴, 크래시 리포트를 조회하고 이슈 정리하는 툴, 공용 슬랙 알림 서비스, 게임 번역 관리 툴 등, 여러가지 게임에 공통으로 사용하는 기능들을 개발하였습니다. 다수의 서비스를 효율적으로 관리하기위해 Spring cloud 환경 기반으로 마이크로 서비스를 지향하는 방식으로 개발 되었고, DB까지 나뉘어진 MSA 환경은 아니었지만 큰 기능 단위로 쪼개어져 유연한 운영이 가능한 구조로 개발하였습니다.

마지막으로 현재 회사에서는 제2의 나라 및 리니지2 레볼루션과 같은 게임 운영툴을 개발하고있습니다. Lagacy Spring 기반의 서비스 및 Spring boot 기반의 신규 서비스, 그리고PHP로 된 Lagacy 또한 같이 유지 보수를 담당했으며, Tableau의 핵심 기능들을 모티브로한 BI 지표 데이터 시각화 툴을 개발하기도 했습니다. 그리고 최근에는 신규 게임 서버 팀 인력 부족으로 인해 게임 서버 개발 업무를 지원하는 역할도 같이 수행하고 있습니다.
